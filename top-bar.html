Top Bar

The gia.ui Top Bar gives you a great way to display a complex navigation bar on small, medium or large screens.

Build with HTML classes

The top bar is a pretty complex piece of magical UI goodness. We rely on many presentational classes to define its look and feel. The top bar is hoverable by default, it will close itself on click by default.

Positioning the Bar

The top bar is build with a singlenavelement with a class of.tb. It will take on full-browser width by default. To make the top bar stay fixed as you scroll, wrap it in adivwith class fixed.f. If you want your navigation to be set to your grid width, wrap it in adivwith class in-grid.ig. You can use.fand.igtogether in the wrapping div (div class="f ig")

Customize with Stylus

Breadcrumbs can be easily customized using our Stylus variables.

Stylus

include-top-bar = true

top-bar-light-background-color = white
top-bar-dark-background-color = oil
top-bar-default-background-color = white
top-bar-border-bottom = 1px solid smoke
top-bar-fixed-z-index = 1

top-bar-height = rem-calc(50)
top-bar-margin-bottom = 0

top-bar-title-font-weight = font-weight-bold
top-bar-title-font-size = rem-calc(22)

top-bar-link-padding-horizontal = 10px
top-bar-link-padding-left = top-bar-link-padding-horizontal
top-bar-link-padding-right = top-bar-link-padding-horizontal
top-bar-link-padding-vertical = 0
top-bar-link-padding-top = top-bar-link-padding-vertical
top-bar-link-padding-bottom = top-bar-link-padding-vertical

Stylus Errors?

Make sure you import this line in main file:

Stylus

@import '_components/top-bar'
