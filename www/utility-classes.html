Utility Classes

gia.ui includes a handful of helpful utility classes to add certain behaviors to your elements.

Float Classes

You can change the float behavior of an element by adding left class.lor right class.rto an HTML element. To clear floats, add the class clearfix.clfto the parent element.

Radius and Round

The radius.raand round.rdclasses allow you to easily apply a border-radius to a UI-element. Adding the class to a button group will apply the border radius only to the outside corners.

Text Align Classes

You can change the text alignment of an element by adding text-left.tl, text-right.tr, text-center.tcor text-justify.tjto an element.

Adding a size to the front of a text alignment class will cause it to only be applied on that size screen or larger (ex. medium-text-center.md-tcwill center text for everything except small screens). Adding a size and only will apply the text alignment just for that media query (ex. medium-only-text-center.mdo-tcwill center text just for medium screens).

Customize with Stylus

Customizing the block grid is easy with the Stylus variables provided in thesettings/utilities.stylfile.

Stylus

include-utilities = true

thumbnail-display = block
thumbnail-border-width = 1px
thumbnail-border-style = solid
thumbnail-border-color = gainsboro
thumbnail-padding-top = 3px
thumbnail-padding-left = thumbnail-padding-top
thumbnail-padding-right = thumbnail-padding-left
thumbnail-padding-bottom = thumbnail-padding-top
thumbnail-background = transparent
thumbnail-default-radius = 3px
thumbnail-radius = 3px
thumbnail-round = round

Stylus Errors?

Make sure you import this line in main file:

Stylus

@import '_components/utilities'
