Typography

Typography in gia.ui is meant to make your life easier by providing clean, attractive, simple default styles for all of the most basic typographical elements.

Headers

gia.ui includes styles for all of the header elements that are balanced and based on a modular scale (though modular-scale is not a dependency of gia.ui).

h1. This is a very large header.


h2. This is a large header.


h3. This is a medium header.


h4. This is a moderate header.


h5. This is a small header.


h6. This is a tiny header.

h3-font-size = rem-calc(17)

h5-font-size = rem-calc(14)
h6-font-size = rem-calc(13)
h1-font-reduction = rem-calc(10)
h2-font-reduction = rem-calc(10)
h3-font-reduction = rem-calc(5)
h4-font-reduction = rem-calc(5)
h5-font-reduction = 0
h6-font-reduction = 0
subheader-line-height = 1.4
subheader-font-color = lighten(header-font-color, 35%)
subheader-font-weight = font-weight-normal
subheader-top-margin = .2rem
subheader-bottom-margin = .5rem
small-font-size = 75%
small-font-color = lighten(header-font-color, 35%)
paragraph-font-family = inherit
paragraph-font-weight = font-weight-normal
paragraph-font-size = 1rem
paragraph-line-height = 1.6
paragraph-margin-bottom = rem-calc(20)
paragraph-text-rendering = optimizeLegibility
code-color = white
code-font-family = font-family-monospace
