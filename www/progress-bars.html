Customize with Stylus

Customizing the visibility classes is easy with the Stylus variables provided in thesettings/visibility.stylfile.

Stylus

include-progress-bars = true

progress-bar-background-color = darken(white, 3)
progress-bar-margin-bottom = include-types ? paragraph-margin-bottom : rem-calc(20)

progress-bar-heights = {}
progress-bar-heights[xsmall-prefix] = rem-calc(3)
progress-bar-heights[small-prefix] = rem-calc(5)
progress-bar-heights[medium-prefix] = rem-calc(8)
progress-bar-heights[large-prefix] = rem-calc(12)
progress-bar-heights[xlarge-prefix] = rem-calc(15)
progress-bar-heights[xxlarge-prefix] = rem-calc(20)

progress-bar-colors = {}
progress-bar-colors[active-class] = active-color
progress-bar-colors[warning-class] = warning-color
progress-bar-colors[error-class] = danger-color
progress-bar-colors[danger-class] = danger-color
progress-bar-colors[info-class] = info-color
progress-bar-colors[success-class] = success-color
progress-bar-colors[disabled-class] = disabled-color
progress-bar-colors[alternate-class] = darken(secondary-color, 10)

progress-bar-radius = radius
progress-bar-round = round
progress-bar-default-color = progress-bar-colors[alternate-class]
progress-bar-default-height = progress-bar-heights[medium-prefix]
progress-bar-default-radius = 0

Semantic Markup with Stylus

You can create your own progress bars using our Stylus mixins.

Basic

You can use thecreate-progress-bar-base()andcreate-progress-bar-meter()mixins to create your own progress bars, like so:

Stylus

.custom-progress-bar-base
  create-progress-bar-base()
  > span.custom-progress-bar-meter
    create-progress-bar-meter()

HTML

Advanced

You can further customize your progress bars using the provided options in thecreate-progress-bar-base()andcreate-progress-bar-meter()mixins:

We can add a border radius to our progress bar through our mixins:

Stylus

.custom-progress-bar-base
  create-progress-bar-base(
    background-color = progress-bar-background-color,
    height = progress-bar-default-height,
    margin-bottom = progress-bar-margin-bottom)
  create-border-radius(size = radius)
  > span.custom-progress-bar-meter
    create-progress-bar-meter(background-color = progress-bar-default-color)
    create-border-radius(size = radius)
