Labels

Labels are useful inline styles that can be dropped into body copy to call out certain sections or to attach metadata. For example, you can attach a label that notes when something was updated.

Regular LabelRadius Active LabelRound Danger LabelWarning LabelSuccess LabelInfo Label

Customize with Stylus

Labels can be easily customized using our Stylus variables.

Stylus

include-labels = true

// Labels
label-display = inline-block
label-cursor = inherit
label-margin-bottom = 0
label-padding-horizontal = rem-calc(7)
label-padding-left = label-padding-horizontal
label-padding-right = label-padding-horizontal
label-padding-vertical = 0
label-padding-top = label-padding-vertical
label-padding-bottom = label-padding-vertical
label-radius = radius
label-default-radius = 0
label-font-family = inherit
label-font-size = rem-calc(12)
label-font-weight = normal
label-font-color-dark = black
label-font-color-light = white
label-line-height = base-line-height
label-border(color = secondary-color)
  border 1px solid darken(color, 3)

Semantic Markup with Stylus

You can create your own labels using Stylus mixins.

Basic

You can use thecreate-label()mixin to create your own labels, like so:

Stylus

.custom-label-class
  create-label()

HTML

...

Advanced

You can further customize your labels using the provided options increate-label()mixin:

Stylus

.custom-label-class
  // Using the available options with default values as shown below.
  create-label(
    margin-bottom = label-margin-bottom,
    padding = label-padding-top label-padding-right label-padding-bottom label-padding-left,
    background-color = secondary-color,
    border-radius = label-default-radius,
    font-size = button-medium-font-size,
    font-weight = button-medium-font-weight,
    font-family = button-font-family,
    line-height = label-line-height
  )

HTML

...
