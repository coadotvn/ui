Semantic Markup with Stylus

You can create your own buttons using our Stylus mixins.

Basic

You can use thecreate-button()mixin like so:

Stylus

.custom-button-class
  // Using the default styles
  create-button()

HTML

...

Advanced

You can further customize your buttons:

.custom-button-class
  // Using the available options with default values as shown below.
  create-button(
    padding = button-medium-padding-vertical button-medium-padding-horizontal,
    background-color = active-color,
    background-hover-color = darken(background-color, button-function-factor),
    border-radius = button-default-radius,
    expanded = false,
    disabled = false,
    font-size = button-medium-font-size,
    font-weight = button-medium-font-weight,
    font-family = button-font-family,
    margin-bottom = button-margin-bottom,
    transition = button-transition,
    transition-duration = button-transition-duration,
    transition-easing = button-transition-easing
  )

HTML

...
